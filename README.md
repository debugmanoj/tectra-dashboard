#  Doctor Management Frontend (React.js + Vite)

This is the **frontend** of the Doctor Management System â€” a responsive web app built with **React.js** and **Vite** that allows users to manage doctors with full CRUD operations.

---

##  Features
- Add, Edit, Delete, and View Doctors  
- Cursor-based pagination  
- Status management (Active / Inactive)  
- Toast notifications via **React Hot Toast**  
- Modular architecture with reusable components  
- Connected to a live backend API deployed on **Vercel**

---

##  Tech Stack
- **React.js (Vite)**
- **Redux Toolkit** for state management  
- **Formik + Yup** for form handling and validation  
- **React Hot Toast** for alerts  
- **Tailwind CSS** for styling
